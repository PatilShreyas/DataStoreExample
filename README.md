# DataStore Example - Android

Jetpack DataStore is a data storage solution. It allows us to store key-value pairs (like SharedPreferences) or typed objects with protocol buffers.
DataStore uses Kotlin and Coroutines + Flow to store data synchronously with consistency and transaction support üòç. In short, it's the new data storage solution which is the replacement of SharedPreferences.
